Congratulations on your new Track 44 email! You are allowed to do whatever you want with it. This is a private server hosted from QuickPacket, running Ubuntu Server 14.04 and Mail-In-A-Box. It is secure, locally hosted, and private.

What can you do?

You can use your browser to access the web client here, or you can add it to almost any mail application or app, but you will need a few settings

Common Mail Settings

Protocol/Method: IMAP

Mail server: mail.track44.moe

IMAP Port: 993

IMAP Security: TLS, otherwise SSL

SMTP Port: 587

SMTP Security: STARTTLS (always)

Username: ALWAYS username@track44.moe

Password: Email password
